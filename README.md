# Socket Chat Sever + GUI Client
## Introduction
This is a basic chat server and client that uses sockets to communicate. The server is written in Python and the client is written in Python with a GUI using PySide6.

## Usage
### Server
The server can be run with the following command:
```bash
python server.py
```

### Client
The client can be run with the following command:
```bash
python client.py
```
